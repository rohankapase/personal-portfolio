# Rohan Kapase | AI & ML Engineer Portfolio

A modern, responsive portfolio website showcasing AI & Machine Learning projects, skills, and professional experience.

## About This Portfolio

This portfolio website is designed to showcase my work as an AI & Machine Learning Engineer. It features a clean, GitHub-inspired dark theme with smooth animations and a fully responsive layout that works seamlessly across desktop, tablet, and mobile devices.

The portfolio includes sections for projects, skills, internships, certificates, about information, and contact. Each project card displays detailed information including technology stack, description, GitHub repository links, and live demo links where available. The skills section is organized into categories such as Programming, Web Technologies, Databases & Tools, Frameworks, Core Concepts, and Soft Skills, making it easy for visitors to understand my technical expertise.

Built using HTML5, CSS3, and vanilla JavaScript, this portfolio is lightweight, fast, and doesn't require any build tools or frameworks. The CSS uses modern techniques including CSS Variables for easy theming, Flexbox and Grid for layout, and smooth transitions for enhanced user experience. The JavaScript handles mobile menu toggle, scroll-based navigation highlighting, and EmailJS integration for the contact form.

The contact section includes a functional form powered by EmailJS, allowing visitors to send messages directly. Social media links to GitHub, LinkedIn, WhatsApp, and Instagram are prominently displayed for easy connectivity. The website is optimized for performance and accessibility, following best practices for web development.
